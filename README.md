Speech_to_text
Who Can Use This App?
Writers: Dictate and transcribe books, autobiographies, or ideas.
Students: Especially helpful for blind or visually impaired students to complete exams or take notes via speech.
Professionals: Transcribe meetings, interviews, or brainstorming sessions efficiently.
General Users: Anyone looking to convert speech into text quickly.
Features
Upload audio files in WAV, MP3, or OGG format.
Transcribes the audio into text using the Hugging Face Wav2Vec2 model.
Simple, easy-to-use interface built with Streamlit.
Future Plans
Add support for real-time transcription via microphone.
Expand to support multiple languages.
Improve error handling for noisy or unsupported audio files.
